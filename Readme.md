# OmniSys


## Overview 
***
### Backend - OmniSys API
>#### Django Rest Framework

OmniSys API is a comprehensive API for applying CRUD operations on various models. 
Its ultimate goal is to provide API endpoints to be used in the frontend application

### UI/UX - OmniOp
